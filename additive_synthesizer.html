Functionality Description

The additive synthesizer generates sound by summing sinusoidal waves at harmonic frequencies. It supports the following features:

  • Harmonics: Allows specifying amplitudes for multiple harmonics.
  • Envelope Generation: Implements ADSR (Attack, Decay, Sustain, Release) envelope shaping.
  • Vibrato: Modulates the pitch with specified rate and depth.

Grading Elements Supported

  • Implement vibrato controlled by the score.
