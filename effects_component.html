Effects Component

Developed by: Rayan Albehair

Functionality Description

The effects component processes audio from the synthesizer by applying an echo effect. It supports the following features:

  • Echo Effect: Adds delayed repetitions of the sound.
  • Effect Parameters: Delay time and wet level can be controlled via the score file.
  • Effect Send Level: Each instrument can specify how much of its signal is sent to the effects processor.

Grading Elements Supported

  • Implement an effects component that processes audio from instruments.
  • Allow the effects to be controlled via the score.
  • Implement effect sends from instruments to the effects processor.
