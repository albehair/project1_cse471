Score File Format

The score files are in XML format and define the musical notes and parameters for the synthesizer. The main elements are:

  • <score>: The root element containing attributes for beats per minute and beats per measure.
  • <effects>: Contains effect definitions that are applied globally.
  • <instrument>: Defines an instrument and its notes.
  • <note>: Defines a note with attributes like measure, beat, duration, and parameters for synthesis.
  • <harmonics>: Contains harmonic amplitudes for the additive synthesizer.
